## API

### DescriptionList

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| list    | 数据列表                                 | DictionObject[]  | [] |

### DictionObject

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| text     | 显示的文字                                 | string  | - |
| value     | 选择后的值                                 | string number boolean  | - |
