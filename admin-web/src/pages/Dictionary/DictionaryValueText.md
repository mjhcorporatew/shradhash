## API

### DescriptionList

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| dataKey    | key config配置的                                 | string  | [] |
| value    | 显示的值                                 | string number  | [] |


### Demo
```jsx harmony
 <DictionaryValueText dataKey="gender" value="1" />
```
